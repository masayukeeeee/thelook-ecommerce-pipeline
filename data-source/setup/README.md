# Data Source Setup

このディレクトリには、プロジェクトで利用するKaggleデータセット(`mustafakeser4/looker-ecommerce-bigquery-dataset`)をダウンロードするためのスクリプトが含まれています。

## 前提条件

- Python 3.x環境
- `uv` がインストールされ、プロジェクトの依存関係がセットアップされていること

## 使い方

以下のコマンドを実行してデータセットをダウンロードします。

```bash
uv run main.py
# または仮想環境を有効化して python main.py
```

ダウンロードされたCSVファイルは `../data/raw/` ディレクトリに保存されます。
